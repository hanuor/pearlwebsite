Pearl
By Hanuor
A powerful image loading, memory caching library for android. With Pearl you can also save Json String and retrieve it later on if you want to display data in your app while the smartphone is in offline mode.

Download


1.Gradle:
compile 'com.github.hanuor:pearl:0.0.4'
2.Maven:
< dependency>
< groupId>com.github.hanuor< /groupId>
< artifactId>pearl< /artifactId>
< version>0.0.4< /version>
< type>pom< /type>
< /dependency>

Usage


Downloading an Image:
Pearl.imageLoader(mContext, URLofImage, imageView);

Saving a Json Object/String:
Pearl.saveJsonObject(Context context, String jsonObject, String tag);

Retrieving a Json Object/String:
Object retrieve = Pearl.retrieveJsonObject(String tag);
String retString = retrieve.toString();
